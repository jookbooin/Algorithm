## 3차원 DP...
상태를 가져야 한다는 것을 인지

1. 시간 t
2. 자두 위치 pos
3. 이동 횟수 mcnt

dp[t][pos][mcnt] 

## 재귀 방식 

#### 1. 점화식 
``` cpp
// 1. 이동 o
go(t + 1, pos^1, m-1);

// 2. 이동 x
go(t + 1, pos, m);
```
#### 2. 재귀
``` cpp
go(int t, int pos, int mcnt){
    if(t == n) return 0; // << t = n일 때 부터 t = 0 일 때까지 +1씩 해서 올라와야 한다/

    int not_move = go(t+1, pos^1, mcnt -1);
    int move = go(t+1, pos, mcnt);

    ret = max(not_move, move) + ( pos == brr[t] - 1);
}
```
